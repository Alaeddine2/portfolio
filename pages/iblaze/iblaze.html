IBlaze Taxi

Taxi and delivery application crafted using Flutter framework, based on new concept and available for multiple countries

Google Play App Store

App Features

IBlaze contains multiple of features:

Order a taxi

User can order a taxi with one click and at anytime.

Book a taxi

User can book a taxi for another time, here the system will choise a taxi randomly then it will link the driver with his passengers

Package delivery

Users can use our drivers [taxis] network to send a package to specific destination

Multiple of payment methods

I have integrate multiple of payment methods like specifly in IBlaze Ivory Coast like: MTN money, Orange payment, EcoBank [SyberSource] and paypal

Application auto-DataBase fragmentation

IBlaze application can separate it data (drivers, users, trips, delivaries...) by partners

Application Concept

IBlaze is a Taxi/Delivery application based on making offers and discussion between users and drivers, it has a very innovative concept that allows user to get best price and service.

IBlaze allow it vehicules network to send packages when they driving to there destination, this solution allows users to send it packages with ship price.

Application sub-features

IBlaze Taxi has two great main fonctionalities [call taxis and packages delivary] but also it has many sub features as:

  • User, driver data collection.
  • Online payment process.
  • Subscribing formular.
  • Developing and hosting two dashboards for app monitoring.
  • OPT validation system.
  • Hosting of OSRM server...
