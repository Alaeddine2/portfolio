IBlaze Taxi

Taxi and delivery application crafted using Flutter framework, based on new concept and available for multiple countries

Google Play App Store

App Features

IBlaze contains multiple of features:

Order a taxi

User can order a taxi with one click and at anytime.

Book a taxi

User can book a taxi for another time, here the system will chose a taxi randomly then it will link the driver with his passenger

Package delivery

Clients can use our drivers [taxis] network to send a package to a specific destination

Multiple of payment methods

I have integrated multiple of payment methods like specifically in IBlaze Ivory Coast like: MTN money, Orange payment, EcoBank [SyberSource] and paypal

Application auto-DataBase fragmentation

IBlaze application can separate its data (drivers, users, trips, delivaries...) by partners

Application Concept

IBlaze is a Taxi/Delivery application based on making offers and discussions between users and drivers, it is very innovative allowing users to get the best price and service.

IBlaze now expnded it's activities to the delivery market, this solution allows users to send their packages with a sheep price.

Application sub-features

IBlaze Taxi has two main fonctionalities [call taxis and packages delivary] but also it has many sub features as:

  • User, driver data collection.
  • Online payment process.
  • Subscribing formular.
  • Developing and hosting two dashboards for app monitoring.
  • OPT validation system.
  • Hosting of OSRM server...
