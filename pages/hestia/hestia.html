App Moduls

WebTrace contains multiple of moduls:

Statistics

This part of project allows admin to get Detailed information about drivers, users, gains...

Real Time Tracer

Admin can track his drivers position, availability, trips in real time using the mobile information.

Account manager

Here the admin can modify, block, remove both of drivers and users account also he has the access to adding new drivers.

Tank tracer

A package of information that allows admin to monitor the application via diplaying the history of trips of drivers paths

Webtrace dahboard

it's a collection of data token from the athor parts of project.

administration

The admin here can control of all application fragment as he can set the coast of trip equation, he can send a push notification to drivers or users, ....

Application Concept

Hestia express administration panel devlopped using Vuejs framework and based on the mobile application database it monotorate and control hestiaExpress application, armed with sockets the panel is fully real time.
